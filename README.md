#Accelerated-Mobile-Pages-express-app
> Get HTML content page from html file to Google AMP (Accelerated Mobile Pages)

## Installation
# clone repo

$ git clone <git repo link>

#  Enter dir
$ cd dir name

# node module install
$ npm install 

# start app now
$ npm start

# hit link
On browser : http://localhost:4300/amp
